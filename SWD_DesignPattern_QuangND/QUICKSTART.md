## ?? Usage Guide

### Factory Method - Create Soft Drink Example
1. Select `1` - Create New Beverage
2. Select `1` - Soft Drink
3. Enter information:
   ```
   Name: Coca Cola
   Price ($): 1.5
   Quantity: 50
   Brand: Coca-Cola
   Volume (ml): 330
   ```
4. ? Soft Drink created successfully!

### Adapter - Create Order Example
1. Select `6` to change payment method (optional)
2. Select `1` - Create New Order
3. Enter information:
   ```
   Customer Name: John Doe
   Select beverage (1-8): 1  (Coca Cola)
   Quantity: 2
   ```
4. ? Order created and payment processed automatically!

### Cancel Order with Refund
1. Select `5` - Cancel Order
2. Enter Order ID
3. Confirm with `y`
4. ? Order cancelled and refunded automatically!
